# Temporary Messenger

A simple one-to-one real-time messaging website built with Node.js, Express, and Socket.IO.

## Features

- Real-time text messaging
- One-to-one chat
- Online user list
- No login system
- No database
- No message history
- Temporary messages only

## Technology Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- Socket.IO
